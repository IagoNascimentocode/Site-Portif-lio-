 ---
##  📁 Como baixar o projeto

```bash

    #Clonar repositorio
    $ git clone https://github.com/IagoNascimentocode/Site-Portifolio

    #Entrar no diretório    
    $ cd Site-Portifolio

    #Instalar dependencias
    $ npm install 

    #Iniciar Projeto
    

```
